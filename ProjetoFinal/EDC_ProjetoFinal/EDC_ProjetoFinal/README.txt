Pastas Importantes:
	Images: Contém todas as imagens necessárias ao projeto;
	Admin: Contém todas as páginas reservadas à parte Administrativa;
	Personal: Contém todas as páginas reservadas à parte privada;
	xml: Contém todos os ficheiros xml gerados bem como o ficheiro xsd responsável da validação do campo xml da base de dados;
	xslt: Contém todos os ficheiros usados na transformação de ficheiros xml;

Ficheiros .sql foram também incluídos para mais fácil visualização do que foi feito relativamente à base de dados;
	- database.sql: Criação da tabela Movies em sql;
	- procedures.sql: Contém todas as stored procedures usadas para o bom funcionamento do sistema;
	- schema.sql: Query responsável pela criação do schema para validação do campo xml da base de dados;

Nota: Para os users classificarem filmes foi usado um sistema de rating baseado num pacote externo "Ajax Control Toolkit".
Penso que não haverá problemas no Professor ver este sistema a funcionar corretamente pois o pacote já está devidamente colocado na "Toolbox" do nosso projeto na pasta
"AjaxToolkit". No entanto, se o Professor por algum motivo não conseguir ver este sistema a funcionar como é suposto, deve proceder à instalação deste pacote, "Ajax Control Toolkit",
através da introdução do seguinte comando no Package Manager Console: "Install-Package AjaxControlToolkit".
